
# 🗳️ Online Voting System

A secure web-based voting platform developed using Django that enables users to register, authenticate, cast votes, and view election results efficiently.

---

## 🚀 Features

- User Registration & Login
- Secure Authentication
- Candidate Management
- Vote Casting
- Real-Time Vote Counting
- Admin Dashboard
- Election Result Management

---

## 🛠️ Tech Stack

### Backend
- Python
- Django

### Frontend
- HTML
- CSS
- Bootstrap

### Database
- SQLite

## ⚙️ Installation

```bash
git clone https://github.com/2200032705-SREEJA/Online-voting-system.git

cd Online-voting-system

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
